# paws-prototype

Idle tactical prototype — **NINE LIVES CORP** operational command UI.

## Stack

- `@paws/core` — pure TypeScript simulation (WASM-ready)
- `@paws/ui` — React + Vite + Canvas map

## Dev

```bash
npm install
npm run dev
```

Open http://localhost:5173 — KOBRA-1 runs missions automatically.

## Test

```bash
npm test
```
